## üéØ Model Mapping Logic

The AI uses keyword-based analysis with confidence scoring:

```python
model_mapping = {
    # Person/Character keywords
    "person": "Man.fbx",
    "human": "Man.fbx", 
    "character": "Man.fbx",
    
    # Animation keywords
    "walking": "Walking.fbx",
    "running": "Running.fbx",
    "idle": "Idle.fbx",
    
    # Technical keywords
    "rig": "Man_Rig.fbx",
    "animation": "Man_Rig.fbx"
}
```

Confidence is calculated based on:
- Number of matching keywords
- Context relevance
- Question complexity
